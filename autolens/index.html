口径说明
车镜 AutoLens · 行业情报 + 用户之声 双数据源口径

行业情报

  • 数据源：懂车帝、汽车之家、盖世汽车、易车、汽车公社、第一电动等公开资讯列表。
  • 五维主题分类：政策 / 市场 / 产品 / 技术 / 舆情。
  • 日更制：每日抓取昨日 00:00-24:00 新闻，规则打分 + Codex 精选生成概要与 Insight。

用户之声

  • 数据源：小红书车型相关用户笔记，懂车帝销量作市场结果参考（不参与反馈综合分）。
  • 覆盖框架：14 品牌 / 32 主销车型，当前仅接入理想真实样本，其余品牌待补充抓取。
  • 查询通道：日常讨论 / 提车反馈 / 避雷吐槽 / 对比选购 / 真实体验，样本量 ≥100 为高置信。
  • 六维评价：动力操控、智能体验、空间舒适、做工质感、品牌认可、价格感知（评分计算未在本样本跑通）。
