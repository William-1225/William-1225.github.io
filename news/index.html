汽车行业新闻每日聚合看板 · 数据流与处理口径

数据源

盖世汽车、汽车之家、第一电动、懂车帝等主流汽车媒体的公开资讯列表。多数源走 Playwright 抓取，反爬较硬的站（如懂车帝）走 curl 抓 SSR 页面；每日运行，礼貌延时，遵守站点 robots。

四维分类

  • 行业大事：政策 / 销量 / 产销 / 财报 / 产业链 / 价格
  • 分品牌：以某车企或品牌动态为主
  • 新能源·智驾：电动化 / 电池 / 智驾 / 智能座舱
  • 舆情·热点：争议 / 维权 / 事故 / 口碑

智能处理

带 LLM 时由大模型（默认 Gemini 2.0 Flash，可回退 OpenAI）输出一句话摘要、分类、品牌标签与重要度；无 key 时走关键词规则兜底。重要度综合来源权重、信号词与品牌命中。

去重与排序

URL 归一指纹去重，seen_ids 跨天去重并标记「新」。列表按重要度降序、时间次之；「今日要点」取重要度 Top 8。
